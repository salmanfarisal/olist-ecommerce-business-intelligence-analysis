# 📊 Olist E-Commerce Business Intelligence Analysis
### End-to-End Data Insights using SQL & R

## 🌟 Overview
This project provides a comprehensive business analysis of the **Olist Brazilian E-Commerce Dataset**. By leveraging **SQL** for complex data extraction and **R** for statistical processing and visualization, this study aims to uncover patterns in consumer behavior, sales performance, and geographic revenue distribution.

## 🛠️ Tech Stack
* **Database:** PostgreSQL
* **Language:** R
* **Key Libraries:** `tidyverse` (ggplot2, dplyr), `DBI`, `RPostgres`

## 📈 Key Business Insights
* **Peak Activity:** Consumer engagement peaks between **10:00 AM and 4:00 PM**.
* **Weekly Trends:** Sales volume is significantly higher on **Mondays and Tuesdays**, suggesting optimal timing for marketing campaigns.
* **Geographic Performance:** While **Sao Paulo** dominates in total revenue, **Divinopolis** records the highest **Average Order Value (AOV)**, indicating a high-spending customer base in that region.
* **Top Categories:** "Cama, Mesa e Banho" and "Beleza e Saude" remain the strongest drivers of order volume.
